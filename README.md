# Vitesse Theme for Zed

A port of the popular [Vitesse theme](https://github.com/antfu/vscode-theme-vitesse) by [@antfu](https://github.com/antfu) for the [Zed editor](https://zed.dev).

Vitesse is a minimal, elegant theme designed for comfortable long-term use with carefully selected colors that are easy on the eyes.

## Installation

### Via Zed Extensions (Recommended - Coming Soon)

1. Open Zed
2. Press `Cmd+Shift+P` (macOS) or `Ctrl+Shift+P` (Linux/Windows)
3. Type "extensions" and select "zed: extensions"
4. Search for "Vitesse"
5. Click Install

### Manual Installation

1. Download `vitesse.json` from this repository
2. Copy it to your Zed themes directory:
   - **macOS/Linux**: `~/.config/zed/themes/`
   - **Windows**: `%APPDATA%\Zed\themes\`
3. Restart Zed
4. Open Settings (`Cmd+,` or `Ctrl+,`)
5. Select your preferred Vitesse variant from the theme picker
